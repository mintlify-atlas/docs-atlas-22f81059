## What is ClansPlus?

ClansPlus is a comprehensive Minecraft plugin that empowers players to create and manage their own clans. Build communities, wage wars, develop skills, and compete for dominance on your server.

## Server Compatibility

ClansPlus supports the latest Minecraft server software:

- **Spigot** - Official support for all recent versions
- **Paper** - Optimized for Paper servers
- **Folia** - Full compatibility with Folia's multi-threaded architecture

<Note>
Requires Java 11 or newer. Spigot forks without compiled NMS code are not supported.
</Note>
