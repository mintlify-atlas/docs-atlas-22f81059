## What is ClansPlus?

ClansPlus is a comprehensive Minecraft plugin that allows players to create and manage their own clans. Build communities, compete in clan wars, upgrade skills, and collaborate with allies in an immersive clan experience.

<Note>
ClansPlus supports Spigot, Paper, and Folia server software with Java 11 or newer.
</Note>

## Core Features

### Clan Operations
- Create and manage clans with customizable names and icons
- Invite and kick members with permission-based controls
- Set custom clan messages/announcements
- Clan spawn points with teleportation
- Private clan chat system
- Ally system for forming alliances with other clans

### Permission System
ClansPlus features a flexible permission system where clan leaders can configure which rank can perform specific actions:
- Invite/Kick members
- Set custom name and icon
- Manage spawn points
- Upgrade clan skills
- Manage allies
- Access clan storage
- Use clan chat

### Skill & Upgrade System
Clans can upgrade various skills to gain advantages:
- **Critical Hit**: Deal increased damage in combat
- **Dodge**: Chance to completely block incoming attacks
- **Life Steal**: Heal when dealing damage
- **Boost Score**: Increase score collection rate

### Clan Wars
Participate in server-wide clan war events:
- Earn points by defeating mobs and players
- Real-time BossBar tracking
- War point rewards for top performers
- MythicMobs support for custom mob scoring

## System Requirements

<Warning>
ClansPlus requires specific server software and Java versions to function properly.
</Warning>

### Supported Server Software
- **Spigot** 1.16+
- **Paper** 1.16+ (recommended for performance)
- **Folia** (fully supported for multi-threaded servers)

<Note>
Spigot forks without compiled NMS code are **not supported**.
</Note>

### Java Requirements
- **Java 11** or newer (required)
- Java 17+ recommended for latest Minecraft versions

### Soft Dependencies (Optional)
These plugins enhance ClansPlus functionality when installed:

- **[Vault](https://www.spigotmc.org/resources/vault.34315/)** - Economy support for clan costs
- **[VaultUnlocked](https://www.spigotmc.org/resources/vaultunlocked.117277/)** - Vault alternative for Folia servers
- **[PlayerPoints](https://www.spigotmc.org/resources/playerpoints.80745/)** - Alternative currency system
- **[PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/)** - Display clan data in chat, scoreboards, and more
- **[MythicMobs](https://www.spigotmc.org/resources/mythicmobs.5702/)** - Custom mob support in clan wars

<Warning>
If running a Folia server, use **VaultUnlocked** instead of Vault to avoid errors.
</Warning>
